# tuner
Simple musical instrument tuner.


## Install

### Dependencies

```bash
sudo apt install libgl1-mesa-dev portaudio19-dev libsdl1.2-dev
```

```bash
pip install 'cython <= 0.28.2'
```

### tuner

```bash
pip install tuner
```

Or from source

```bash
cd path/to/tuner
pip install -e .
```

Then run:

```bash
tuner
```

to start the application.
